## 插件安装

下载对应版本安装包，进入网页，点击下载，获取xpi文件

![xpi文件](public/pic/1732248320429.png)

打开zotero进入  工具-附加组件-**Install Add-on From Files**-选择下载的xpi文件进行安装

![插件安装](https://pic-go-42.oss-cn-guangzhou.aliyuncs.com/img/20231215111835.png)
